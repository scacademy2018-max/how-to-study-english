前提

英語学習の基本的な考え方

英語は兎にも角にも単語力が命です。単語力がなければ、文法も長文も得点は安定しません。帰国子女や海外留学経験があるような特殊な場合を除き、英語は英単語学習で語彙力をつけつつ、文法・構文・長文へと学習を進めていくのが定石です。
📝
単語力が最優先
どんな得意科目でも、語彙力がなければ文法・長文で安定した得点は望めません。まず単語から。
⚡
短期集中で一気に覚える
毎日10個ずつでは定着に1年超かかる場合も。1日100〜200個のペースで集中して覚えましょう。
🎯
単語→文法→構文→長文の順に
この順序が最短ルート。偏差値に関わらず基本の流れは同じです。

Level 01
DEVIATION VALUE
偏差値 45 未満
【現状】 中学範囲の抜けがかなり大きい段階です。特に英単語と文法がかなり抜けていると考えられます。そのため、3か月を目処に中学範囲の総復習を行い、基本的な中学単語と中学英文法を覚え直しましょう。

学習方針

英単語と文法を下記の参考書から1冊ずつ選んで学習してみてください。「中学英語を〈もう一度〉はじめからていねいに」「中学英語をもう一度ひとつひとつわかりやすく。」「カラー改訂版 世界一わかりやすい中学英語の授業」あたりは問題数が少ないため、「中学校３年間の英語が１冊でしっかりわかる問題集」などで演習量を補ってください。

これだけで一気に偏差値50まで上がるということはありませんが、基本が押さえられるだけで以降の参考書学習がとてもスムーズになります。土台をしっかり固めることが、この段階での最優先事項です。

Level 02
DEVIATION VALUE
偏差値 45 〜 55
【現状】 高校1〜2年生くらいの基礎事項が抜けている可能性が高いです。進学校ではやや難しめの単語帳が渡されることが多いですが、無視してOK。自分のレベルに合った単語帳を使うのがベストです。

学習方針

「英単語ターゲット1200」や「システム英単語Basic」などで基礎的な単語力を固めつつ、「大岩のいちばんはじめの英文法【超基礎文法編】」で基本的な高校英文法を習得するのがおすすめです。「大岩〜」が終わったら、「関正生の英文法ポラリス［1 標準レベル］」まで手を伸ばすとよいでしょう。

英文法と単語の学習が終わったら、英文解釈に進んでください。偏差値が50に満たない状態だと「入門英文解釈の技術70」はやや苦しいかもしれませんが、苦しいと感じるということはポラリス1のやりこみが甘いサイン。全然わからないと思ったらポラリス1に戻りましょう。
DO ✓
  • 自分のレベルに合った単語帳を選ぶ
  • 文法→解釈の順で進める
  • 詰まったら文法書に戻る
DON'T ✗
  • 学校配布の難しい単語帳に固執する
  • 文法が曖昧なまま長文に進む

Level 03
DEVIATION VALUE
偏差値 55 〜 60
【現状】 英文法はある程度安定してきた段階。共通テストのみ使用の場合はここで共通テスト対策にシフトして問題ありません。二次試験や私大入試でも使う場合は、入試問題に合わせて対策を変えていきましょう。ここでは一般的に英文法・長文の両方が出題されると想定して話を進めます。

学習方針

偏差値55以上になると英文法はある程度安定していると思います。入試でさらに高いレベルが求められるなら、「関正生の英文法ポラリス［2 標準レベル］」まで手を伸ばすとよいでしょう。英文解釈では「英文読解の特別講座」や「英語長文Rise 構文解釈１」まで手を伸ばしてみてもよいと思います。

あとは基本的な長文問題で読解練習を積み、ある程度読み慣れてきたら読解方法を体系的に学ぶのがおすすめ。「パラグラフリーディングのストラテジー１」を読み、読解の方法を体系的に学習しましょう。単語帳は「ターゲット1400」を軸に、志望校のレベルに応じて「システム英単語」や「ターゲット1900」にまで手を伸ばしてみましょう。

Level 04
DEVIATION VALUE
偏差値 60 超
【現状】 難関大を目指せる実力の段階。共通テストのみの人は過去問に集中し、自由英作文など大学ごとの対策に注力しましょう。ここでは旧帝大（東大・京大・阪大あたりを除く）や東京科学大・一橋大などの難関大を除く、難関大学を目指す方向けに解説します。

学習方針

読解力にイマイチ自信がなく、その原因が構文にあるなら「英語長文Rise 構文解釈１」に戻るか、それが理解できている段階なら「ポレポレ英文読解プロセス50」や「難関大に合格する英文解釈Code70」がおすすめです。そのうえで「関正生の英語長文ポラリス[2 応用レベル]」など標準〜やや難関向けの問題集で演習を繰り返しましょう。

ここでやってはいけないのが、難しすぎる長文問題集を読みまくることです。帰り読みのクセがついてしまったり、変な読み方のクセがついてしまったり、復習で知識を吸収しきれなかったりと、学習効率が著しく低下します。あくまで自分に合ったレベルの問題を丁寧に進めましょう。

語彙力に不安があれば「システム英単語」や「ターゲット1900」を完璧にすることを目指しましょう。熟語で躓くなら「英熟語ターゲット1000」もおすすめです。難関大と言っても、難しいことばかりすればよいということではありません。基礎を再度確認し、抜け漏れをなくすことが合否を左右します。
DO ✓
  • 自分のレベルに合った長文問題集を繰り返す
  • 基礎の抜けがないか再確認する
  • 語彙・熟語を完璧にする
DON'T ✗
  • 難しすぎる問題集を読みまくる
  • 帰り読みのクセをつける
